# PWA 설치 방법

[English](./pwa.md) | 한글

> [!NOTE]
> * PWA(Progressive Web App)는 웹 기술을 기반으로 웹 브라우저에서 설치할 수 있는 애플리케이션 유형입니다.
> * 일반적인 웹사이트와 달리 PWA는 네이티브 앱의 장점을 활용하면서도 웹의 편리함을 제공합니다.
> * PWA는 설치와 삭제가 쉽고, 자동으로 업데이트되며, 네이티브 앱 수준의 사용자 경험을 제공합니다.
